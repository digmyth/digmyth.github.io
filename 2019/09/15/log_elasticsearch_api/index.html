知识点

搜索引擎包括： 索引组件、搜索组件
索引组件： 面向数据存储和索引构建
搜索组件： 面向用户提供搜索功能以及将用户提供的搜索请求转换成可用的查询语句并通过索引完成查询过程或搜索过程
其中一种著名的索引叫做倒排索引

获取数据的组件：工作在pull拉取数据模式， solr/Nutch/grub/Apeture
ES: 索引index、类型type、文档document、映射mapping
每个索引的分片数量： 5
每个分片也应该有副本： 1
用户接口： 9200/tcp
集群接口： 9300/tcp

数据类型： string、number、boolean、date、
