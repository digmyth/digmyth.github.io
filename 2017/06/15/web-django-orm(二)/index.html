1.4 总结

正向查询用字段（这里字段是指外键字段）

  • all: 字段+.跨表字段
  • values: 字段 + __跨表字段
  • values_list 字段+ __跨表字段
    反向查询用表名
  • all: 表名小写+_set
  • values: 表名小写 + __跨表字段
  • values_list 表名小写+ __跨表字段

总结一句话就是： 正向查询用字段，反向查询用表名
